CABLE NAMING
C2F / 003 / f12 / u4
• C2F = cable direction and segment
• 003 = cable serial number
• f12 = total fiber capacity
• u4 = used fibers
• Signal direction is always from core node to access node
Examples:
• O2D = OLT → ODF
• D2C = ODF → CLS
• C2F = CLS → FAT
• F2S = FAT → SUB
